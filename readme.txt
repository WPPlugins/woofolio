WooFolio comes with WooCommerce Filterable Product with Image Slider. Filter Products animations in a beautiful way like WooFolio.

== Description ==

Filter Products animations in a beautiful way like Portfolio ( We Called it WooFolio ). An awesome way to show your WooCommerce Products on Filtering system. We have given the option with Product Images Sliders also. If you forgot to add Product Images then by default Featured Image will be taken for Thumbnail images. 
To Show WooFolio you'll need a shortcode [jwt_woofolio] , Create a New Page and place the Shortcode. WooFolio will appear.

== Installation ==
WooFolio Installation process is like - you'll need a shortcode [jwt_woofolio] , Create a New Page and place the Shortcode. WooFolio will appear.

1. From WordPress Dashboard go to "Plugins>> Add New>> Uploads", select 'woofolio.zip' file and upload it. Or,
2. Upload 'woofolio.zip' to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4 To show all Products a shortcode [jwt_woofolio] needed, Create a New Page and place the Shortcode. WooFolio will appear.
